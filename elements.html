QuickHire

Below is a 5 Minute video of the QuickHire Team and I detailing the project.

I speak for the first 2 minutes detailing the Data Science Team contributions to the QuickHire Stakeholders. QuickHire was a greenfield project with a team of 10 Engineers. There were 3 Data Scientist, 5 Web Deveoplers and 1 UX/UI designer. We also had 1 project manager and 2 stakeholders.

Project was completed in 6 weeks and laid the ground work for future integrations.

Data Science Stack

  • Python
Web Scraper
  • Beautiful Soup, Selenium
Hosting
  1. Postgres DB, on a AWS RDS
  2. AWS Elastic Beanstalk instance
  3. Flask API, Hosted on a separate AWS Elastic Beanstalk
  4. Scraper are ran by a aws lambda function every 12 hours Models are run every 7 mins by the same aws function
Models
  1. Latent Dirichlet Allocation (LDA) Model
  2. KNearest Neighbors
  3. Flask API, Hosted on a separate AWS Elastic Beanstalk
